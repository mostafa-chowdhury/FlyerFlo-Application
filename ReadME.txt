+FlyerFlo-Application
+====================
+
+Program that downloads Reddit alien image to a local folder from given Web Page URL.

How to run the program:
  1. Run program as Java Application.
  2. Dialog box will prompt for URL of the Reddit page you wish to download the image from.
  3. If URL is invalid, the program will close.
  4. If URL is valid, then name the file (default name is "image.jpg") and choose the folder to save to.
     Note: Extension must be .jpg as I didn't account for other formats.
  5. File will be saved in the specified folder.
  6. Dialog message will show confirmation message with path to the image file before program closes.
